# Weather Dashboard (No API)

A simple weather dashboard built using HTML, CSS, and JavaScript without using any external API.

## Features
- No API key required
- Weather data loaded from local JSON file
- Clean and responsive UI
- Instant search
- Perfect for resume and GitHub portfolio

## Tech Stack
- HTML
- CSS
- JavaScript
- JSON (local data)

## How to Run
1. Download or clone the repository.
2. Open `index.html` directly in your browser.

## Deployment (GitHub Pages)
1. Create a new GitHub repository.
2. Upload all project files.
3. Go to **Settings → Pages**.
4. Choose:
   - Branch: `main`
   - Folder: `/root`
5. Save.

Your live project link will appear in 10–20 seconds.
